⚙ 首次設定 — Firebase API Key

讀取學生答題紀錄需要 Firebase API Key。
Key 只存在你的瀏覽器，不會上傳至任何伺服器。

  1. 開啟 Firebase Console → 選你的專案
  2. 左下角「專案設定」（齒輪圖示）
  3. 「您的應用程式」→ 找到 apiKey
  4. 複製貼入下方
Firebase API Key
Key 格式：AIzaSy... （約 39 字元）
